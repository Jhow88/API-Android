Dispositivos Móveis

Questões:
  1. Defina dispositivos Móveis

    2. Resposta:

    Dispositivos móveis são aparelhos eletrônicos portáteis que possuem capacidade de processamento, conectividade e interação com o usuário. Eles são projetados para serem transportados e usados em movimento

  2. Qual o papel da internet no atual avanço tecnológico?

    3. Resposta:

    A Internet possibilita hoje uma difusão rápida, através das novas tecnologias de informação, do conhecimento permitindo às empresas não só melhorar a sua eficiência mas fundamentalmente oferecerem novos produtos e serviços pelos quais os consumidores, através dos mecanismos de mercado, manifestam a sua preferência.

  3. Cite os principais tipos de dispositivos móveis

    4. Resposta:

    Os principais dispositivos móveis são: smartphone, ultrabook, notebook, netbook, GPS, tablet.

  4. Para que serve o nível de aplicação de Arquitetura Android?

    5. Resposta:

    Construída de forma modular, essa API de alto nível implementa diversas funcionalidades como a criação de views, gerenciamento de recursos, notificações e alertas de usuário, navegação entre os aplicativos

  5. O SO(sistema operacional) do Android é baseado no Linux. Qual seu papel na arquitetura do Android?

    6. Resposta:

    Linux é um kernel (núcleo), ou seja, parte central de um SO responsável pelo gerenciamento de tarefas de baixíssimo nível. Grosso modo, é ele que faz a comunicação do software com o hardware — componentes e periféricos do computador, como memórias, processador e placa de vídeo.

  6. Qual a principal característica do Android?

    7. Resposta:

    Android é um sistema operacional (SO) baseado no núcleo Linux, projetado principalmente para dispositivos eletrônicos móveis (como smartphones e tablets) com tela sensível ao toque ou interface de usuário baseada na manipulação direta

  7. Qual a diferença de um aplicativo nativo para um desenvolvido por terceiros?

    8. Resposta:

    Os aplicativos nativos são capazes de proporcionar aos usuários uma experiência mais intensa do que os híbridos. Porque eles são desenvolvidos especialmente para funcionar em determinado sistema operacional e oferecer melhores recursos de hardware, facilitando a utilização de funções e recursos próprios do app

  8. Como o android se diferencia de outros sistemas de dispositivos móveis?

    9. Resposta:

    Ele é feito sob a tutela do open-source, o que permite que qualquer um modifique, seja o usuário ou companhias

  9. Como funciona o nível de framework da arquitetura do Android?

    10. Resposta:

    Provê um conjunto de bibliotecas para acessar os diversos recursos do dispositivo como interface gráfica, telefonia, localizador (GPS), banco de dados persistente, armazenamento no cartão SD, etc.

  10. E o nível de bibliotecas

    11. Resposta:

    Os niveis são 4 camadas: Applications -Application FrameWork - Libraries - Android RunTime - Linux kernel.
